# **Math Game**

![Math Game](/og-images/projects/math-game.jpg)

## **Overview**

An interactive and educational math game built with Next.js and TypeScript. This project is designed to make learning math fun and engaging for users of all ages.

- Multi-step game flow and dynamic game logic
- Points system and persistent user progress
- Server Side Rendering with Next.js for fast performance
- Responsive and accessible interface
- Strong type safety with TypeScript throughout the codebase
